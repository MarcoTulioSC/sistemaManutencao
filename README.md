# sistemaManutencao


Sistema interno desenvolvido pelo apps script(via planilha de dados sheets), para automatização de problemas no setor de manutenção, na empresa Ansal, no qual é direcionado os dados do web app(ao digitar, pegando os de um sistema de logística da empresa chamado Transnet) para a planilha sheets, automaticamente.
